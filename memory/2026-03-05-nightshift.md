# 2026-03-05 夜班学习记录

## 学习点 1: MCP (Model Context Protocol)
- **时间**: 2026-03-07 01:34
- **内容**: Anthropic 于 2024年11月推出的开放协议，用于将 AI 助手与外部数据源和工具无缝集成
- **来源**: https://github.com/modelcontextprotocol, https://modelcontextprotocol.io/specification/2025-11-25
- **用途**: 解决 AI 应用与信息孤岛、遗留系统的集成问题

## 学习点 2: Playwright 浏览器自动化
- **时间**: 2026-03-07 01:34
- **内容**: Microsoft 2020年推出的开源 Node.js 库，支持 Chrome/Firefox/WebKit 统一 API，包含自动等待、内置测试运行器、Trace Viewer
- **来源**: https://www.browserstack.com/guide/playwright-vs-puppeteer
- **对比**: vs Puppeteer - Playwright 跨浏览器支持更好，Puppeteer 适合 Chrome 专用/需要 CDP 深度访问的场景

## 学习点 3: No-code AI 工作流工具
- **时间**: 2026-03-07 01:34
- **内容**: 2026年主流 no-code AI 自动化平台包括 Vellum、Kissflow、Noloco 等，46%的产品团队已使用 AI 驱动工作流
- **来源**: https://www.vellum.ai/blog/no-code-ai-workflow-automation-tools-guide, https://noloco.io/blog/best-workflow-automation-tools
- **用途**: 非技术人员也能构建 AI 驱动的自动化流程

## 学习点 4: n8n 工作流自动化平台
- **时间**: 2026-03-07 01:34
- **内容**: Fair-code 许可的工作流自动化平台，400+集成，原生 AI 能力，可自托管或云端，支持可视化构建+自定义代码(JavaScript/Python)
- **来源**: https://github.com/n8n-io/n8n
- **用途**: 自主托管工作流，保持数据完全控制

## 学习点 5: Taskfile (taskfile.dev)
- **时间**: 2026-03-07 02:34
- **内容**: Go 编写的任务运行器，用 Taskfile.yml 描述命令和工作流，比 Makefile 更易读，支持变量、模板、依赖任务、跨平台
- **来源**: https://taskfile.dev/, https://dev.to/anthonymax/8-developer-tools-that-will-boost-your-workflow-in-2026-3p8k
- **用途**: 替代 Makefile，实现更清晰可维护的构建/测试/部署自动化

## 学习点 6: GitHub CLI (gh)
- **时间**: 2026-03-07 02:34
- **内容**: GitHub 官方命令行工具，支持创建 PR、review、issue 管理、run workflow、gist、repo 操作，无需浏览器
- **来源**: https://cli.github.com/, https://www.qodo.ai/blog/best-cli-tools/
- **用途**: 在终端完成 PR review、合并、检查状态，CI/CD 集成

## 学习点 7: Lazygit
- **时间**: 2026-03-07 02:34
- **内容**: Go 编写的终端 Git UI，键盘驱动，支持可视化 rebase、stage、stash、commit amend，提供 staged/unstaged diff 视图
- **来源**: https://github.com/jesseduffield/lazygit
- **用途**: 复杂 Git 操作（rebase、cherry-pick）的可视化替代，比纯命令行更高效

## 学习点 8: FFmpeg 多媒体处理
- **时间**: 2026-03-07 02:34
- **内容**: 开源多媒体框架，命令行处理音视频（转码、剪辑、提取、压制、串流），几乎支持所有格式
- **来源**: https://ffmpeg.org/, https://www.qodo.ai/blog/best-cli-tools/
- **用途**: 视频压缩、格式转换、批量处理、提取音频、录制屏幕
