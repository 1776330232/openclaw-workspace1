# 定时检查任务（用户发消息时触发）

## 待办事项提醒规则

每天只提醒一次，待办事项：
1. 读取 ~/.openclaw/tasks.json
2. 读取 ~/.openclaw/last-reminder.json 检查是否已经提醒过
3. 如果有待办 + 今天还没提醒 → 提醒用户
4. 提醒后更新 last-reminder.json 的日期

## 提醒话术

早上提醒：
"早上好 Hang！今天有 X 个待办事项：
1. 去吃鱼汤
2. 清空淘宝日用品购物车
3. 拿出脚垫下的红包
4. 收拾房间清理垃圾

加油完成它们！🐕"

## 重置规则

- 每天 0 点自动重置提醒状态（日期变化时）
- 或者用户说"重新提醒待办"时重置
